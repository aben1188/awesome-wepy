### :point_right: 月付20K以上找小程序/小游戏兼职程序员或外包团队
- **报酬：** 报酬可按周支付，同时可按收入分成，收入上不封顶
- **要求：**
	- 实际开发过从前端到后端完整的小程序(包括微信支付或红包功能)或小游戏的个人和团队；
	- 有社交红包类开发经验者或H5页游/手游开发经验者优先；
	- 能力出众、经验丰富的开发者，以上优先条件可忽略。
- **不看学历和履历，只看资历和能力**
